# SEC-II-2026 大作业交付物仓库

CampusHub 是一个校园服务平台，仓库同时包含前端、后端和阶段性交付文档。

## 项目基本情况

- 前端：Vue 3 + TypeScript + Vite + Pinia
- 后端：Spring Boot 3.5 + MyBatis Plus + MySQL
- 默认后端端口：8080
- 默认前端开发端口：5173
- 前端通过 Vite 代理访问后端接口，开发时无需手动改接口地址
- 已加入测试管理员账号'ADMIN001'，密码'Admin123!'

## 目录说明

- `frontend/`：前端源码
- `backend/`：后端源码
- `docs/`：各阶段文档与交付物
- `start.sh`：一键启动脚本

## 启动方式

只需要在仓库根目录执行下面这一个命令即可同时启动前后端：

```bash
./start.sh
```

脚本会自动启动后端和前端，启动后可以直接访问前端开发页面。

## 环境要求

- JDK 21
- Node.js 18 或更高版本
- npm
- MySQL 可用，并确保本地数据库已按照`backend\数据库配置快速指南（团队成员必读）.md`配置好
- 若要自己注册账号，请确保本地环境变量已按照`backend\邮箱验证码发信配置说明.md`配置好
