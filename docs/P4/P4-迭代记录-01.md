## 1. 公共基础层

- `common/api/ApiResponse`
- `common/api/PageResponse`
- `common/exception/BusinessException`
- `common/exception/ErrorCode`

- `common/model/PageQuery`

## 2. 用户模块骨架

已新增：

- `auth/domain/User`
- `auth/domain/UserRole`
- `auth/domain/UserStatus`
- `auth/dto/RegisterCommand`
- `auth/dto/LoginCommand`
- `auth/dto/LoginResult`

- `auth/repository/UserRepository`
- `auth/repository/InMemoryUserRepository`
- `auth/service/AuthApplicationService`
- `auth/service/AuthApplicationServiceImpl`

- `auth/service/NoopVerificationCodeService`

说明：

- 当前仓储为内存实现，仅用于第一轮规则开发和测试验证
- 后续切换 MyBatis 时，优先替换 `UserRepository` 实现，不动服务层主流程

## 3. 密码加密依赖

已在 `backend/pom.xml` 中补充：

- `org.springframework.security:spring-security-crypto`

用途：

- 使用 `BCrypt` 完成密码哈希，避免明文存储

## 4. 已落实业务规则

本轮已经写入代码并通过测试的规则包括：

- 学号唯一性校验
- 邮箱唯一性校验
- 密码最小长度校验
- 注册密码使用 `BCrypt` 哈希
- 登录支持 `studentId` / `email` 作为 `loginId`
- 封禁用户不能登录
- 用户只能修改自己的资料
